The Problem: What if we don't have clean data?


  • So far we assumed access to clean training data $x \sim p(x)$

  • But what if we only have noisy observations $y = Ax + n$?

  • Real-world scenarios:
    • Medical imaging (always has measurement noise)
    • Astronomy (atmospheric effects, detector noise)
    • Many other domains where clean data is unavailable

Challenge
How can we learn diffusion priors when we only have access to corrupted data?

DiEM: Learning Diffusion Priors from Observations

Rozet et al. (2024)


Formulation as Empirical Bayes
  • Learn prior $p_\theta(x)$ from noisy observations $y_i$
  • Use Expectation-Maximization (EM) algorithm

EM Algorithm Structure:

  • E-step: Sample from posterior $p(x|y)$ using MMPS
  • M-step: Train denoiser on samples

Key Insight
Produces "proper" diffusion models that can be used for any downstream task

DiEM Algorithm


Iterative Process:

  1. Initialize with Gaussian prior
  2. E-step: Sample from posterior using current prior
  3. M-step: Update prior by training on samples
  4. Repeat until convergence
